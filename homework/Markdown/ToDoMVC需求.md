# ToDoMVC需求

1. 任务列表
    - 有数据的时候展示出来。
    - 没有数据的时候隐藏 main 部分。

2. 添加任务
    - 页面初始化获得焦点。
    - 敲回车添加到任务列表。
    - 不允许有非空数据。
    - 添加完成后清空文本框。

3. 标记所有任务完成或者未完成

4. 任务项
    - 可切换任务完成状态。
    - 可删除单个任务项。
    - 双击label进入编辑模式。

5. 编辑任务项
    - 编辑文本框自动获得焦点。
    - 在编辑文本框敲回车后者失去焦点后，如果为空，则直接删除这个item，如果不为空，保存这个数据，并去除editing样式。
    - 输入状态按下esc取消编辑。

6. 其他（footer部分）
    - 显示所有未完成任务数。
    - 清除所有的已完成任务。
    - 将数据持久化到localStorage中。
    - 路由状态切换。
